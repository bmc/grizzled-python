Class Hierarchy

  • optparse.OptionContainer: Abstract base class.
    • optparse.OptionParser: Class attributes: standard_option_list : [Option] list of standard options that will be accepted by all instances of this parser class (intended to be overridden by subclasses).
      • grizzled.cmdline.CommandLineParser: Custom version of command line option parser.
  • ConfigParser.RawConfigParser
    • ConfigParser.ConfigParser
      • ConfigParser.SafeConfigParser
        • grizzled.config.Configuration: Configuration file parser.
  • zipfile.ZipFile: Class with methods to open, read, write, close, list zip files.
    • grizzled.io.Zip: ``Zip`` extends the standard ``zipfile.ZipFile`` class and provides a method to extract the contents of a zip file into a directory.
  • object: The most base type
    • grizzled.io.AutoFlush: An ``AutoFlush`` wraps a file-like object and flushes the output (via a call to ``flush()`` after every write operation.
    • exceptions.BaseException: Common base class for all exceptions
      • exceptions.Exception: Common base class for all non-exit exceptions.
        • ConfigParser.Error: Base class for ConfigParser exceptions.
          • ConfigParser.NoOptionError: A requested option was not found.
          • ConfigParser.NoSectionError: Raised when no section matches a requested option.
        • grizzled.exception.ExceptionWithMessage: Useful base class for exceptions that have a single exception message argument.
          • grizzled.db.base.DBError: Base class for all DB exceptions.
            • grizzled.db.base.Error: Thrown to indicate an error in the ``db`` module.
            • grizzled.db.base.Warning: Thrown to indicate an error in the ``db`` module.
          • grizzled.history.HistoryError: Thrown to indicate history errors, when another exception won't do.
          • grizzled.file.includer.IncludeError: Thrown by ``Includer`` when an error occurs while processing the file.
          • grizzled.config.NoVariableError: Thrown when a configuration file attempts to substitute a nonexistent variable, and the ``Configuration`` object was instantiated with ``strict_substitution`` set to ``True``.
          • grizzled.misc.ReadOnlyObjectError: Thrown by ``ReadOnly`` to indicate an attempt to set a field.
        • exceptions.StandardError: Base class for all standard Python exceptions that do not represent interpreter exiting.
          • exceptions.EnvironmentError: Base class for I/O related errors.
            • exceptions.OSError: OS system call failed.
              • grizzled.os.DaemonError: Thrown by ``daemonize()`` when an error occurs while attempting to create a daemon.
    • grizzled.db.base.Cursor: Class for DB cursors returned by the ``DB.cursor()`` method.
      • grizzled.db.dbgadfly.GadflyCursor
    • grizzled.db.base.DB: The object returned by a call to ``DBDriver.connect()``.
      • grizzled.db.dbgadfly.GadflyDB
    • grizzled.db.base.DBDriver: Base class for all DB drivers.
      • grizzled.db.dummydb.DummyDriver: Dummy database driver, for testing.
      • grizzled.db.dbgadfly.GadflyDriver: DB Driver for Gadfly, a pure Python RDBMS
      • grizzled.db.mysql.MySQLDriver: DB Driver for MySQL, using the MySQLdb DB API module.
      • grizzled.db.oracle.OracleDriver: DB Driver for Oracle, using the cx_Oracle DB API module.
      • grizzled.db.postgresql.PostgreSQLDriver: DB Driver for PostgreSQL, using the psycopg2 DB API module.
      • grizzled.db.sqlserver.SQLServerDriver: DB Driver for Microsoft SQL Server, using the pymssql DB API module.
      • grizzled.db.sqlite.SQLite3Driver: DB Driver for SQLite, using the pysqlite DB API module.
    • grizzled.db.dummydb.DummyCursor
    • grizzled.db.dummydb.DummyDB
    • grizzled.net.ftp.parse.FTPListData: The `FTPListDataParser` class's ``parse_line()`` method returns an instance of this class, capturing the parsed data.
    • grizzled.net.ftp.parse.FTPListDataParser: An ``FTPListDataParser`` object can be used to parse one or more lines that were retrieved by an FTP ``LIST`` command that was sent to a remote server.
    • grizzled.net.ftp.parse.FTPMlstDataParser: An ``FTPMlstDataParser`` object can be used to parse one or more lines that were retrieved by an FTP ``MLST`` or ``MLSD`` command that was sent to a remote server.
    • grizzled.io.filelock.FileLock: A ``FileLock`` object models a file lock.
    • logging.Formatter: Formatter instances are used to convert a LogRecord to text.
      • grizzled.log.WrappingLogFormatter: A ``logging`` ``Formatter`` class that writes each message wrapped on line boundaries.
    • grizzled.proxy.Forwarder: The ``grizzled.forwarder.Forwarder`` class is intended to be used as a mixin, to make it easier for classes to forward calls to another class.
    • grizzled.history.History: Base class for history implementations.
      • grizzled.history.DummyHistory
      • grizzled.history.ReadlineHistory
        • grizzled.history.PyReadlineHistory
    • grizzled.file.includer.Includer: An ``Includer`` object preprocesses a path or file-like object, expanding include references.
    • grizzled.collections.dict.LRUList
    • grizzled.collections.dict.LRUListEntry
    • grizzled.io.MultiWriter: Wraps multiple file-like objects so that they all may be written at once.
    • grizzled.io.PushbackFile: A file-like wrapper object that permits pushback.
    • grizzled.misc.ReadOnly: A ``ReadOnly`` object wraps another object and prevents all the contained object's fields from being written.
    • string.Template: A string class for supporting $-substitutions.
      • grizzled.config._ConfigTemplate: Subclass of string.Template that handles our configuration variable reference syntax.
    • grizzled.io.filelock._PosixFileLock: File lock implementation for POSIX-compliant systems.
    • grizzled.io.filelock._WindowsFileLock: File lock implementation for Windows systems.
    • dict: dict() -> new empty dictionary dict(mapping) -> new dictionary initialized from a mapping object's (key, value) pairs dict(iterable) -> new dictionary initialized as if via: d = {} for k, v in iterable: d[k] = v dict(**kwargs) -> new dictionary initialized with the name=value pairs in the keyword argument list.
      • grizzled.collections.dict.LRUDict: ``LRUDict`` is a dictionary of a fixed maximum size that enforces a least recently used discard policy.
      • grizzled.collections.dict.OrderedDict: ``OrderedDict`` is a simple ordered dictionary.
      • grizzled.config._ConfigDict: Dictionary that knows how to dereference variables within a parsed config.
    • tuple: tuple() -> empty tuple tuple(iterable) -> tuple initialized from iterable's items
      • grizzled.db.base.IndexMetadata: IndexMetadata(index_name, index_columns, description)
      • grizzled.db.base.RDBMSMetadata: RDBMSMetadata(vendor, product, version)
      • grizzled.db.base.TableMetadata: TableMetadata(column_name, type_string, max_char_size, precision, scale, nullable)
    • type: type(object) -> the object's type type(name, bases, dict) -> a new type
      • string._TemplateMetaclass
